# Forge Reality

Forge Reality is a web-based utility platform that provides tools for processing digital content, extracting insights, and interacting with media from the web.

The project focuses on building practical automation tools accessible through a lightweight browser interface.

---

## Overview

Forge Reality is designed as a modular platform containing tools for processing online content and extracting useful information.

The system combines Python backend utilities with a browser-based interface to allow users to interact with multiple tools from a single place.

---

## Features

• Highlight extraction engine for textual content
• Context generation tools for structured information
• Smart YouTube media downloader utilities
• Web interface with templates and dynamic UI
• Modular Python backend architecture
• CSS animations and interactive frontend behavior

## Running Locally

### Clone Repository

```
git clone https://github.com/YOUR_USERNAME/forge-reality
cd forge-reality
```

---

### Install Dependencies

```
pip install -r requirements.txt
```

---

### Run Application

```
python app.py
```

Open in browser:

```
http://localhost:5000
```
